# Web-chat-project
Weaving connections in real time bespoke python and html web-chat application, a first year project with professional polish.

## Description
A simple real-time chat application built python, flask and html showcasing a clean backend.

## Installation
First make sure you have python installed.
Install the Flask framework using this command:pip install flask

## Features
Real-time chat application.

## Usage
To start the application just type this command in your terminal:app.py
